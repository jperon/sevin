\couplet{
	Vous qui preniez sur vos genoux\\*
	Les tout-petits de chez vous, \vbis\\
	Vous avez pris nos Petits-Loups :\\*
	Jésus, soyez-leur très doux. \vbis
}

\couplet{
	Soyez très doux à leurs cheftaines\\*
	Souvent mortes à la peine,\\
	Pour que leurs mains, de grâces pleines,\\*
	Bénissent leurs sizaines.
}

\couplet{
	Vous avez pris, clairs adolescents,\\*
	Nos scouts de treize et seize ans.\\
	Séchez de vos doigts apaisants\\*
	Les pleurs de leurs parents.
}

\couplet{
	Seigneur Jésus, nos grands Routiers\\*
	Allaient à vous tout entiers ;\\
	Faites au bout de leurs sentiers\\*
	Accueil à nos Routiers.
}

\couplet{
	Seigneur Jésus, de nous les meilleurs\\*
	Sont morts en vrais sauveurs ;\\
	Ah ! donnez-leur place d'honneur\\*
	Tout près de votre Cœur.
}

\couplet{
	Pour tous nos chefs, pour nos prêtres morts,\\*
	Seigneur, nous prions encor ;\\
	Pour nous s'usaient tous, âme et corps :\\*
	Priez pour tous nos morts.
}

\couplet{
	Nous vous prions pour nos amis,\\*
	Pour nos très chers ennemis,\\
	Qu'un jour soyons tous réunis\\*
	En votre Paradis.
}
