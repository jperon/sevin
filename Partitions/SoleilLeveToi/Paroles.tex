\couplet{%
    La route qu'on suit\\*
    Le front dans la nuit\\
    Jamais ne conduit\\
    Une démarche sûre :\\
    Le pied se débat,\\
    Il fait des faux-pas,\\*
    Se lasse et n'avance pas :
}

\refrain{%
    Soleil, lève-toi, remplis les cieux,\\*
    Soleil, lève-toi, remplis nos yeux :\\
    Le monde n'est beau\\
    Que sous ton flambeau,\\*
    Remplis-le d'un jour nouveau !
}

\couplet{%
    L'air vif du matin\\*
    Réveille l'entrain\\
    Et les gais refrains\\
    Qui font vibrer nos âmes :\\
    La nuit disparaît,\\
    Vidant la forêt\\*
    Des ombres qui la peuplaient !
}

\couplet{%
    On voit l'horizon,\\*
    Les champs, les sillons,\\
    Le toit des maisons\\
    Se fondre dans la brume.\\
    Le coq a chanté,\\
    La cloche a tinté,\\*
    Le jour est ressuscité !
}
