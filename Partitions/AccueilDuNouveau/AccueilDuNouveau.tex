\refrain{%
	Frère loup dans notre sizaine,\\*
	Frère loup, sois le bienvenu.%
}

\couplet{%
	Pattes tendres et vieux loups,\\*
	Tu pourras compter sur nous!%
}

\couplet{%
	Avec nous, fais de ton mieux\\*
	Pour ouvrir tout grand les yeux.%
}

\couplet{%
	Des vieux loups suivant les traces,\\*
	Tu feras de grandes chasses.%
}

\couplet{%
	La sizaine est dans la joie :\\*
	Elle sait qu'on peut compter sur toi.%
}
