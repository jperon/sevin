\refrain{%
	Bonsoir à Valloires\\*
	Où la paix du ciel tient tous les cœurs unis ;\\
	Bonsoir à Valloires,\\*
	Les jours qu'on y passe à jamais sont bénis.
}

\couplet{%
	Ô Saint Bernard dont les fils ont vécu dans ce lieu,\\*
	Daignez bénir les enfants qu'y reçoit le Bon Dieu.
}

\couplet{%
	Ô Notre-Dame, modèle de tous dévouements,\\*
	Récompensez celles qui sont ici des mamans.
}

\couplet{%
	Seigneur Jésus, qui régnez sur la sainte Abbaye,\\*
	Protégez ceux par qui fut notre Troupe accueillie.\\*
	\emph{Merci à Valloires…}
}

\couplet{%
	Demain verra le départ matinal des campeurs.\\*
	Nous reviendrons dans ces murs qui nous rendent meilleurs.\\*
	\emph{Adieu, cher Valloires…}
}
