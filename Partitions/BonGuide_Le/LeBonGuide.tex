\couplet{%
	Tu montras dans l'étable\\*
	Un Dieu pauvre aux pasteurs !\\
	Vers les biens véritables\\*
	Guide tes Éclaireurs.
}

\couplet{%
	Nous voyons pleins d'envie\\*
	L'Enfant-Dieu sur ton cœur :\\
	Vers le Christ de l'Hostie\\*
	Guide tes Éclaireurs.
}

\couplet{%
	Charpentier doux et sage,\\*
	Patron des travailleurs,\\
	Au labeur de leur âge,\\*
	Guide tes Éclaireurs.
}

\couplet{%
	Un regard sur Marie\\*
	Te payait tes sueurs ;\\
	Dis-nous comme on la prie,\\*
	Guide tes Éclaireurs.
}

\couplet{%
	Protecteur de la Vierge,\\*
	Chasse les tentateurs ;\\
	Quand le mal les submerge,\\*
	Guide tes Éclaireurs.
}

\couplet{%
	À notre heure dernière,\\*
	Enlève ses terreurs,\\
	Au Pays de Lumière,\\*
	Guide tes Éclaireurs.
}
